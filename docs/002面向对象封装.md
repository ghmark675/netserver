# 目录结构说明

从此节开始，项目采用以下目录结构：
- `include`:头文件
- `src`:源文件
- `obj`:目标文件
- `bin`:生成的可执行文件
- `docs`:项目文档
- `test`:测试目录

由于只能有一个`main()`，我们可以考虑在程序运行的参数中添加例如`-c`或`-s`的选项来决定此次运行的是客户端还是服务端
